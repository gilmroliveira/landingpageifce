História dos Sistemas Operacionais

1969
Unix
Criado por Ken Thompson e Dennis Ritchie nos Bell Labs. Tornou-se a base para muitos sistemas operacionais modernos, incluindo Linux e macOS.
1985
Windows 1.0
Primeira versão do sistema operacional Windows da Microsoft, introduzindo uma interface gráfica baseada em MS-DOS.
1991
Linux
Linus Torvalds lança o kernel Linux como projeto open-source. Hoje é a base do Android, servidores web, supercomputadores e sistemas embarcados.
1995
Windows 95
Revolucionou a computação pessoal com seu menu Iniciar e interface aprimorada. Vendido 7 milhões de cópias nas primeiras 5 semanas.
2008
Android
Sistema operacional móvel desenvolvido pelo Google, baseado no kernel Linux. Atualmente o SO mais usado no mundo em dispositivos móveis.
2021
Windows 11
A versão mais recente do Windows, com redesenho da interface e melhor integração com dispositivos Android.
